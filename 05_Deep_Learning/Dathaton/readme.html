Quais as chances de você entrar em uma universidade?

A proposta do projeto é um estudo de algoritmos de predição com o objetivo de prever a probabilidade dos alunos da Passos Mágicos de conseguir uma bolsa de estudos na universidade. A solução traz junto de si uma explicação dos parametros utilizados para que o leitor não tenha dúvidas de como a solução funciona.

1. Introdução

Este projeto é um aplicativo de previsão de bolsas usando um modelo Random Forest. Abaixo você encontrará informações sobre como instalar, configurar e usar o aplicativo.

1.1. Sobre os Paramêtros

Os parâmetros de configuração para cada modelo foram os seguintes:

1.2. Sobre os Modelos

Utilizando os dados disponibilizados pela escola referente aos alunos matriculados no ano de 2022, Investigamos a distribuição desses alunos com base em suas aptidões nas áreas de estudo e classificações que a própria escola disponibiliza.

A base foi separada em teste e treino, sendo utilizada 80% para treino e 20% para teste. Os modelos utilizados no estudo foram os:
